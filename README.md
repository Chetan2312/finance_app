# finance_app
personal finance management app, using local data and storing data locally.
 to launch:
1. The simplest way with no dependencies:

```bash
cd /home/chetan/Desktop/personal/finance_app
python3 -m http.server 8080
```

Then open http://localhost:8080 in your browser.

2. Alternatively, if you have Node.js:

```bash
npx serve /home/chetan/Desktop/personal/finance_app
```

Or 

npx live-server for auto-reload on file changes:

```bash
npx live-server /home/chetan/Desktop/personal/finance_app
```
